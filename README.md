# Scroll-to-Cart

Motor de conteúdo de vídeo curto para lojas online. Decide **que conteúdo
produzir** antes de gastares tempo e dinheiro, e gera os guiões prontos.

Companion do guia gratuito *Scroll-to-Cart* da [datascript.ch](https://datascript.ch).

## O que está aqui

| Ficheiro | Para quê |
|---|---|
| `SKILL.md` | O prompt mestre. As regras de produção dos guiões. |
| `templates/01-contexto-produto.md` | Ensina à IA o que é o teu produto. |
| `templates/02-referencias.md` | Mostra à IA o que já funciona no teu nicho. |

## Como usar

1. Preenche os dois ficheiros em `templates/`. Não atalhes o contexto.
2. Abre um projeto no Claude (ou ChatGPT). Carrega o `SKILL.md` nas instruções
   do projeto e os dois templates preenchidos como ficheiros de contexto.
3. Pede, num chat novo:
   > "Gera 5 guiões em formato [demo / unboxing / antes-depois / review] com
   > hook do tipo [identidade / dor / curiosidade / contraste]."
4. Pontua cada output /30. Escala 24+, ajusta 21 a 23, mata abaixo de 21.
5. A cada 2 semanas, mete os vencedores nas referências e aperta o prompt.

## O Teste das 3 Perguntas

Antes de produzir qualquer formato:

1. **Pára o polegar?** (atenção)
2. **Aguenta 50 sem falir?** (escala)
3. **Leva à loja ou ao like?** (conversão)
